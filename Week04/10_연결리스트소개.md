# 연결 리스트 (Linked List)

## 한방향 vs 양방향

### 한방향 리스트
![단방향리스트](/images/한방향리스트.png)  
- 한 노드에 key값 한 개와 link 한 개를 가진다.  

<br>

```python
class Node:
    def __init__(self, key=None):
        self.key = key
        self.next = None
    
    def __str__(self):
        return str(self.key)

a = Node(3)
b = Node(9)
c = Node(-1)
a.next = b
b.next = c
c.next = None # 
```

```
#실행결과
a(3) --> b(9) --> c(-1) --> None
```
- 개별적으로 만들면, 노드마다 변수를 할당해야 함
- 모든 노드에 변수를 할당하는 것이 아닌 헤드노드만 할당, 나머지는 헤드노드로 부터 접근가능  

<br>

```python
class SinglyLinkedList:
    def __init__(self):
        self.head = None
        self.size = 0
    + 삽입 메소드
    + 삭제 메소드
    def __len__(self):
        return self.size
    
```
